DotaKeys версия 0.2
===================

1. ВВЕДЕНИЕ
-----------

DotaKeys - это набор AutoIt скриптов для переназначения клавиш. Также доступны функции 
создания макросов и автоклик мышью.

2. УСТАНОВКА
------------

Для запуска DotaKeys вам необходимо скачать интерпретатор языка AutoIt с официального сайта:
http://www.autoitscript.com/site/autoit/downloads

Если вы используете 64-разрядную версию Windows, выберите 64-разрядную версию интерпретатора
в диалоге установки.

Скачайте архив со скриптами DotaKeys и распакуйте его:
https://github.com/ellysh/DotaKeys/archive/master.zip

3. НАСТРОЙКА
------------

3.1 ПЕРЕНАЗНАЧЕНИЕ КЛАВИШ
-------------------------

Клавиши для переназначения указаны в скриптах каталога `run`. Для изменения этих клавиш
откройте файл с именем нужного героя в любом текстовом редакторе и измените переназначение.

Пример переназначения клавиш для героя Admiral:
	global const $kKeys = "s,h|q,e|w,d|e,x|r,t"

Согласно этой строчке будут выполняться следующие переназначения клавиш:
	s -> h
	q -> e
	w -> d
	e -> x
	r -> t

Нажатие одной клавиши может приводить к последовательному срабатыванию нескольких клавиш.
Например:
	global const $kKeys = "x,q,w,e|s,h"

В данном примере при нажатии клавиши `x` скриптом буду последовательно нажаты клавиши `q`, `w` и `e`.
Кроме того нажатие клавиши `s` будет перекладываться в `h`.

3.2 АВТОКЛИК
------------

Функция автоклика позволяет эмулировать левый клик мыши в текущей позиции курсора после
нажатия любой клавиши.

Механизм автоклика включается в файле `config\global.au3`:
	global const $kIsAutoClick = false

При значении false автоклик выключен, при true - включен.

3.3 СИСТЕМНЫЕ КЛАВИШИ
---------------------

Для приостановки работы DotaKeys скриптов используется клавиша `ENTER`. Вместо этой клавиши
можно указать любую другую в файле `config\global.au3`:
	global const $kPauseKey = Chr(13)

Пример для назначении клавиши `PAUSE` вместо `ENTER`:
	global const $kPauseKey = "{PAUSE}"

4. ЗАПУСК
---------

ВНИМАНИЕ: Скрипты DotaKeys это не исполняемые exe файлы! Все запускаемые скрипты с расширением
au3 расположены в каталоге `run`.

Для запуска DotaKeys выполните следующие действия:
1. Запустите скрипт из каталога `run` с именем нужного героя.
2. Переключитесь на окно Warcraft 3/Dota 2.
3. Нажмите Alt+F2 для запуска скрипта.
4. Для завершения работы скрипта нажать Alt+F1.
